流程控制

流程控制语句, 用于设定计算执行的次序, 简历程序的逻辑结构

流程控制语句一般起以下三个作用:

  • 选择
  • 循环
  • 跳转

Go 语言中支持如下几种流程控制语句:

  • 条件语句: if, else 和 else if
  • 选择语句: switch, case 和 select
  • 循环语句: for 和 range
  • 跳转语句: goto

为了能在具体的场景中满足更丰富的控制需求, Go 语言还增加了如下关键字: break, continue 和 fallthrough

一. 条件语句

对应关键字: if, else, else if

需要注意有以下几点:

  • 条件语句不需要用括号将条件包含起来
  • 无论语句体内有几条语句, 花括号都是必须存在的
  • 在 if 之后, 条件语句之前, 可以添加变量初始化语句, 用 ; 号间隔
package main

import "fmt"

func example(x int) {
	if x == 0 {
		fmt.Println("x == 0")
	} else {
		fmt.Println("x is", x)
	}
}

func main() {
	var x = 10
	example(x)
}

二. 选择语句

对应关键字: switch, case, select

需要注意有以下几点:

  • 条件表达式不限制为常量或者整数
  • 单个 case 中, 可以出现多个结果选项
  • 只有在 case 中明确添加 fallthrough 关键字, 才会继续执行紧跟的下一个 case
package main

import "fmt"

func main() {
	i := 3

	switch i {
	case 1:
		fmt.Printf("1")
	case 2:
		fallthrough
	case 3, 4, 5:
		fmt.Printf("3, 4, 5")
	default:
		fmt.Printf("Default")
	}
}
  • 可不设定 switch 之后的条件表达式
package main

import "fmt"

func example(num int) {
	switch {
	case 0 <= num && num <= 3:
		fmt.Println("0-3")
	case 4 <= num && num <= 6:
		fmt.Println("4-6")
	}
}

func main() {
	a := 5

	example(a)
}

三. 循环语句

对应关键字: for, range

Go 中的循环语句只支持 for 关键字

需要注意有以下几点:

  • Go 的 for 循环条件中允许定义和初始化变量, 但 Go 不支持以逗号为间隔的多个赋值语句, 必须用平行赋值的方式来初始化多个变量
  • Go 的 for 支持 continue 和 break 来控制循环, 而且还能够指定中断哪一个循环.
sum := 0

for i := 0; i < 10; i++ {
	fmt.Println(sum)
}

// 无限循环的场景
sum := 0

for {
	sum++
	if sum > 100 {
		break
	}
}

// 条件表达式中也支持赋值
a := []int{1, 2, 3, 4, 5, 6}
for i, j := 0, len(a) -1; i < j; j = i + 1, j -1 {
	a[i], a[j] = a[j], a[i]
}

for j := 0; j < 5; j++ {
	for i := 0; i < 10; i++ {
		if i > 5 {
			break JLoop
		}
		fmt.Println(i)
	}
}

四. 跳转语句

对应关键字: goto

goto 的作用就是跳转到本函数内的某个标签.

func myfunc() {
	i := 0
HERE:
	fmt.Println(i)
	i++
	if i < 10 {
		goto HERE
	}
}
