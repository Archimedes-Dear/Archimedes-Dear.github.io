Golang 安装和环境配置

2019年10月8日
| Golang
| 阅读

安装

Golang 配置方式分为两种:

  • Go Modules包管理模式
  • 原始GOPATH模式

前提:

  1. Golang 官网
  2. Golang 国内官网
  3. GoModules配置 思否 技术文

一. 包管理模式

该模式从1.12版本开始支持, 需要在环境变量中设置 GO111MODULE=auto 或 GO111MODULE=on

若要开启国内代理加速的话, 需要配置环境变量 GOPROXY=https://goproxy.cn 该加速由七牛云提供, 另一个加速地址为 https://goproxy.io

然后从官网下载 Golang-1.12.x 版本(或更高版本), 进行安装. 然后配置GOPATH, GOROOT环境变量, 分别指向项目包路径和 Golang 安装路径即可.

二. GOPATH模式

从官网下载 Golang-1.12.x 版本(或更高版本), 进行安装. 然后配置GOPATH, GOROOT环境变量, 分别指向项目路径和 Golang 安装路径即可.

其中, GOPATH为项目路径, 里面分为3个子文件夹, 分别是:

  • bin 存放 go install 生成的程序
  • pkg 存放编译后的依赖库
  • src 存放项目源码

两种安装模式推荐第一种安装模式, 使用 GOPATH 的模式当有多个项目和依赖时, 使用起来不方便
